Format for using this program should be as follows

- Output redirection "COMMAND >output_file"
	output_file could be what ever you want the file name to be.
	Note no space between > and the output file.
	
	For example "ls -a -l >kome.txt"

- Pipe call should be "COMMAND |SECONDCOMMAND"
	Note there is no space between the | and the SECONDCOMMAND

	Example "cat kome.txt |less"

- This program is allowed in every command. The options for the file reading is removed in this question
as question 4 was modified not question 5
